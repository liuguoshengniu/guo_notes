对于 R2S（基于 aarch64 架构）运行的 iStoreOS，下载最新版 Passwall 或 Passwall2 插件及其依赖包，最推荐的三个官方/主流渠道如下：

1. iStoreOS 官方插件包 (最推荐)
这是针对 iStoreOS 优化的 .run 安装包，集成了依赖，安装最简单。

下载地址： GitHub - AUK9527/Are-u-ok (Apps 目录)

版本说明： 截至 2026 年 1 月，提供有 PassWall_25.12.27 和 PassWall2_25.12.25 等版本。

安装方法： 下载对应的 .run 文件后，在 iStoreOS 后台的“iStore 应用商店”选择“手动安装”，上传文件即可。

2. Passwall 官方 Release 频道 (手动更新)
如果你需要最纯净、最新的编译版本，直接去开发者的发布页面。

Passwall (经典版)： GitHub - Openwrt-Passwall/openwrt-passwall/releases

Passwall2 (重构版)： GitHub - Openwrt-Passwall/openwrt-passwall2/releases

下载建议： * 下载 luci-app-passwall_xxx_all.ipk（主程序）和 luci-i18n-passwall-zh-cn_xxx_all.ipk（汉化包）。

注意： 手动安装 .ipk 可能提示缺少依赖（如 xray-core、sing-box 等），需要一并下载对应架构（R2S 选择 aarch64_cortex-a53 或 aarch64_generic）的组件包。

3. OpenWrt-Passwall-Build (第三方自动构建)
这是一个自动同步官方源码并编译好的镜像站，适合寻找特定构架的依赖包。

SourceForge 下载站： openwrt-passwall-build

路径： 依次进入 snapshots -> packages -> aarch64_cortex-a53 -> passwall_packages，可以找到最新的 Xray、Sing-box、Shadowsocks 等核心组件。

💡 安装建议
架构确认： R2S 属于 aarch64 架构，下载组件包时请认准该标识。

Passwall vs Passwall2： * Passwall： 经典稳定，功能成熟。

Passwall2： 对较新的协议（如 Hysteria2、TUIC）支持更好，界面略有不同。

依赖问题： 如果手动安装 ipk 失败，建议先在 SSH 执行 opkg update，或优先使用第一种方式提供的 .run 全家桶。

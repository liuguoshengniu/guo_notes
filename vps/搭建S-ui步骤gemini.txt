## 第二步：申请 SSL 证书（针对 Hysteria2, Tuic, Vmess）

你需要为 `rn1.356765.xyz` 申请证书。我们使用 `acme.sh`。

1. **安装 acme.sh**:
```bash
curl https://get.acme.sh | sh
alias acme.sh=~/.acme.sh/acme.sh

```


2. **申请证书**（请确保 Cloudflare 的解析生效，且暂时关闭小黄云以通过验证，或使用 API 方式。此处推荐最简单的 Standalone 模式，需确保 80 端口未被占用）：
```bash
acme.sh --upgrade --auto-upgrade
acme.sh --set-default-ca --server letsencrypt
acme.sh --issue -d rn1.356765.xyz --standalone

```


3. **安装证书到指定目录**:
```bash
mkdir -p /root/cert
acme.sh --install-cert -d rn1.356765.xyz --key-file /root/cert/private.key --fullchain-file /root/cert/cert.crt

```


*注意：对于 Hysteria2 和 Tuic 的子域名，你可以重复此步骤或直接使用泛域名证书。*
